# 50-30-20 Budget Planner
A hyper-minimalist financial tool to instantly allocate your monthly income based on the 50/30/20 rule.

## 🚀 Concept: 1 Tool / 1 Action
Stop tracking every cent. Start managing your future. This tool focuses on "Allocation" rather than "Recording," helping you make instant financial decisions without the friction of complex apps.

## ✨ Features
- **Zero Friction:** No login. No tracking. No ads. Just a tool.
- **Currency Agnostic:** Works with any currency unit ($, €, ¥, RM, etc.).
- **Smart Advice:** Minimal guidance to keep your budget balanced.
- **Tap to Copy:** Instantly copy calculated results to your clipboard.
- **Privacy First:** All calculations are done locally in your browser.

## 📐 The 50/30/20 Rule
- **50% Needs:** Essential expenses (Rent, Food, Utilities).
- **30% Wants:** Lifestyle choices (Dining out, Hobbies).
- **20% Savings:** Future-proofing (Investments, Debt repayment).

## 🛠 Tech Stack
- Pure HTML / CSS / JavaScript (Optimized for Mobile Safari & Chrome).
- 15px Rounded Corner UI Design.
